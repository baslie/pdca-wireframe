# PDCA Wireframe

Design handoff пакет для корпоративного сайта PDCA Consulting — дизайн-токены, React-компоненты, TypeScript-интерфейсы и инструкции для реализации.

## Что внутри

- **Дизайн-система** — CSS custom properties (цвета, шрифты), конфигурация Tailwind CSS v4
- **React-компоненты** — готовые TSX-компоненты для всех секций сайта
- **TypeScript-интерфейсы** — контракты данных для UI
- **Инструкции** — пошаговые гайды для coding-агентов (one-shot и посекционный варианты)
- **Промпты** — шаблоны промптов для автоматизированной реализации

## Секции сайта

| Секция | Описание |
|--------|----------|
| Shell | Хедер, футер, навигация |
| Главная страница | Лендинг с hero, преимуществами и CTA |
| Каталог тренингов | Список тренингов с фильтрацией |
| Страница тренинга | Детальная страница отдельного тренинга |
| Корпоративные программы | Консалтинг и корпоративные решения |
| Клиенты и кейсы | Портфолио и истории успеха |
| Команда и контакты | О компании, тренеры, контактная информация |

# One-Shot Implementation Prompt

Скопируйте этот промпт и передайте coding-агенту вместе со всем содержимым папки `product-plan/`.

---

## Промпт

Ты — опытный фронтенд-разработчик. Тебе предстоит реализовать корпоративный сайт **PDCA Consulting** — консалтинговой компании, специализирующейся на бережливом производстве и повышении эффективности бизнеса.

### Контекст

Тебе предоставлен дизайн-пакет в папке `product-plan/` со следующими материалами:

- `product-overview.md` — описание продукта, секции, сущности, дизайн-система
- `design-system/tokens.css` — CSS custom properties (цвета, шрифты, отступы)
- `design-system/tailwind-colors.md` — конфигурация цветов Tailwind CSS v4
- `design-system/fonts.md` — подключение шрифтов Google Fonts
- `instructions/one-shot-instructions.md` — полные инструкции по всем этапам
- `instructions/incremental/01-shell.md` ... `07-komanda-o-nas-kontakty.md` — детальные инструкции
- `data-shapes/` — TypeScript-интерфейсы для каждой секции
- `shell/` — компоненты оболочки (хедер, футер, навигация)
- `sections/` — компоненты секций с готовой вёрсткой

### Задача

Реализуй полный сайт, следуя этапам из `instructions/one-shot-instructions.md`. Выполняй этапы последовательно:

1. **Shell** — Sticky-хедер с навигацией, mega-footer, мобильное меню
2. **Главная страница** — Hero, сегментация, направления, доверие, кейсы, FAQ, CTA
3. **Каталог тренингов** — Каталог с фильтрацией по категориям
4. **Страница тренинга** — Landing page отдельного тренинга
5. **Корпоративные программы** — Программы с дорожными картами
6. **Клиенты и кейсы** — Портфолио с фильтрацией и детальными страницами
7. **Команда, О нас и Контакты** — Профили, история, контакты

### Технические требования

Перед началом работы задай уточняющие вопросы по:
- Фреймворку (Next.js / Remix / Vite + React Router / другой)
- Управлению состоянием и data fetching
- Бэкенду и API (headless CMS, REST, GraphQL)
- Деплою и хостингу

Затем реализуй, учитывая:

- **Tailwind CSS v4** — не создавай `tailwind.config.js`, используй `@theme` директиву
- **Шрифт Manrope** — подключи через Google Fonts с subset `cyrillic`
- **Фирменные цвета**: primary `#e84249` (coral-red), secondary `#003154` (dark-blue), neutral slate
- **Адаптивная вёрстка**: mobile-first, breakpoints — `sm:`, `md:`, `lg:`, `xl:`
- **Props-based компоненты**: все данные через props, никаких hardcoded значений
- **TypeScript**: используй интерфейсы из `data-shapes/`
- **Формы**: валидация обязательных полей, callbacks для отправки данных
- **Accessibility**: семантический HTML, ARIA-атрибуты, keyboard navigation

### Результат

После реализации каждого этапа подтверди:
- Компонент рендерится без ошибок
- Адаптивная вёрстка работает (mobile / tablet / desktop)
- Интерактивные элементы функционируют (фильтры, аккордеоны, слайдеры, формы)
- Навигация между секциями работает

# Опросник для клиента PDCA Consulting

> Этот документ содержит список вопросов, ответы на которые необходимы для наполнения сайта реальным контентом. Сейчас сайт — это wireframe с заглушками (вымышленные имена тренеров, выдуманные кейсы, условные метрики и контактные данные). Для запуска полноценного сайта нужна реальная информация по всем разделам.

---

## Блок 1. Компания — общая информация

**1.** Как точно называется компания? Полное и сокращённое наименование (сейчас используется «PDCA Consulting» / ООО «ПДКА Консалтинг» — это верно?).

**2.** В каком году основана компания? (сейчас указан 2012 год)

**3.** Кто основатель компании? Имя, фамилия и краткая история создания.

**4.** Миссия компании — как бы вы сформулировали миссию в 1–2 предложениях? (сейчас: «Мы помогаем российским предприятиям достигать операционного совершенства через внедрение проверенных методологий непрерывных улучшений и развитие культуры бережливого мышления»)

**5.** Описание компании для раздела «О нас» — 3–5 предложений о том, чем занимается компания, для кого, в чём уникальность подхода.

**6.** Методология — используете ли вы цикл PDCA (Plan-Do-Check-Act) как основу? Если да, опишите ваш подход в 2–3 предложениях. Если у вас другая методология — опишите её.

**7.** Ключевые метрики компании — укажите актуальные цифры:
   - Сколько лет на рынке?
   - Сколько проектов реализовано?
   - Сколько корпоративных клиентов?
   - Сколько специалистов обучено?
   - Другие значимые цифры (например, география присутствия, количество тренеров и т.д.)

**8.** История компании / ключевые вехи — перечислите 4–6 значимых событий в истории компании (год + что произошло). Например: расширение команды, получение сертификации, юбилейный проект, выход на новый рынок и т.д.

**9.** Сертификаты и аккредитации компании — перечислите все актуальные сертификаты, аккредитации и членства в ассоциациях. Для каждого укажите:
   - Название сертификата
   - Кем выдан
   - Год получения
   - Есть ли логотип в электронном виде? (если да — приложите позже)

   - Управление и Лидерство
   - Персональная эффективность

   Это верный список? Нужно что-то добавить, убрать или переименовать?

**17.** Для каждого направления — напишите 1 предложение, описывающее ключевой результат для клиента (например: «Рост производительности, снижение потерь и брака»).

**18.** В каких форматах вы работаете? Сейчас в wireframe три формата:
   - Корпоративные тренинги (на площадке клиента, группа 8–20 чел.)
   - Открытые программы (сборные группы из разных компаний)

   Это актуально? Нужно что-то скорректировать? Опишите каждый формат в 1–2 предложениях.

**19.** Для каких отраслей вы работаете? Перечислите основные отрасли ваших клиентов. Сейчас в wireframe:

   - Ритейл и сервис

   Полный ли это список? Для каждой отрасли укажите: типичную «боль» клиентов и ваше решение (по 1 предложению).

## Блок 4. Каталог тренингов

**20.** Полный список тренингов — пожалуйста, предоставьте список всех тренингов/программ. Для каждого тренинга нужна следующая информация:
   - Название
   - К какому направлению относится (из списка из вопроса 16)
   - Краткое описание (2–3 предложения)
   - Длительность (например: «16 часов / 2 дня»)
   - Формат: офлайн, онлайн, гибридный
   - Целевая аудитория (руководители, линейный персонал, офисные сотрудники, все)
   - Ключевой результат (1 предложение — что получит участник)

**21.** Для 3–5 самых востребованных тренингов предоставьте расширенную информацию:
   - **Для кого этот тренинг** — опишите 3–4 типичные роли участников и их типичные проблемы/запросы (например: «Начальник цеха — не умеет выявлять потери на участке»)
   - **Результаты в цифрах** — 3–4 измеримых результата (например: «Снижение брака на 20–40%», «Рост OEE на 10–15%»)
   - **Программа тренинга** — перечислите модули/блоки тренинга. Для каждого модуля: название + 3–5 тем
   - **Кто проводит** — имя тренера (из списка из вопроса 23)
   - **Стоимость** — цена за группу или за участника. Если цена «по запросу» — так и укажите
   - **FAQ** — 3–5 частых вопросов и ответов, специфичных именно для этого тренинга

## Блок 6. Команда / Тренеры

**23.** Список тренеров и консультантов — для каждого члена команды предоставьте:
   - **ФИО** полностью
   - **Должность / роль** (например: «Ведущий тренер-консультант, эксперт по Lean»)
   - **Краткое био** (2–3 предложения для карточки на главной странице)
   - **Полное био** (2–3 абзаца для персональной страницы тренера)
   - **Специализации** — список из 3–5 направлений (например: «Бережливое производство», «VSM», «SMED»)
   - **Сертификаты и регалии** — список всех сертификатов, степеней, членств (например: «Lean Six Sigma Black Belt (ASQ)», «MBA, Управление производством»)
   - **Опыт** — сколько лет в отрасли
   - **Какие тренинги проводит** — список тренингов (из вопроса 20) с указанием, сколько раз проведён каждый (если такая статистика есть)
   - **Ключевые достижения / highlights** — 2–3 факта для блока на главной странице (например: «Провёл 200+ проектов», «Сертифицированный Lean Six Sigma Black Belt»)

**24.** Фотографии тренеров — нужны профессиональные портретные фотографии каждого тренера. Требования:
   - Формат: JPG или PNG
   - Минимальный размер: 400×500 px (вертикальная ориентация)
   - Желательно: единый стиль фото (деловой портрет)
   - *Понимаю, что фото могут быть предоставлены позже. Укажите примерные сроки.*

## Блок 7. Клиенты и кейсы

**25.** Список клиентов для раздела «Клиенты» — перечислите компании, которые можно указать как ваших клиентов. Для каждого:
   - Название компании
   - Отрасль

   - Есть ли логотип в электронном виде (SVG, PNG)? (может быть предоставлен позже)

**26.** Кейсы (реализованные проекты) — это один из ключевых разделов сайта. Для каждого кейса нужна следующая информация:
   - **Название проекта** (например: «Lean-трансформация обогатительной фабрики»)
   - **Клиент** — название компании (из списка вопроса 25)
   - **Отрасль** и **направление услуги** (из списков вопросов 19 и 16)
   - **Задача / проблема** — что за проблема была у клиента? (3–5 предложений)
   - **Решение** — что вы сделали? (3–5 предложений)
   - **Результаты в цифрах** — 3–4 ключевых метрики (например: «Сокращение потерь: −40%», «Обучено: 120 чел.»)
   - **Этапы проекта** — 3–5 этапов, каждый: название + описание в 1 предложение
   - **Есть ли фотографии с проекта?** (тренинг в процессе, участники, площадка и т.д.) — *могут быть предоставлены позже*

**27.** Сколько кейсов вы готовы предоставить? Рекомендуем минимум 5–7 кейсов из разных отраслей и направлений для наполненности раздела.

**28.** Отзывы клиентов — для каждого кейса (или отдельно) предоставьте отзыв:
   - Текст отзыва (3–5 предложений)
   - ФИО автора
   - Должность автора
   - Название компании
   - *Есть ли письменное согласие на публикацию отзыва?*
